### Health Score Formula:

```
Health Score (0-100) =
  + Login frequency (30 points)
  + Active tests (25 points)
  + Tests completed (20 points)
  + Support satisfaction (15 points)
  + Payment status (10 points)
```

### Scoring Details:

**Login Frequency (30 pts):**
- 30: Daily
- 20: 2-3x/week
- 10: Weekly
- 5: Biweekly
- 0: < Monthly

**Active Tests (25 pts):**
- 25: 3+ active tests
- 15: 2 active tests
- 10: 1 active test
- 0: 0 active tests

**Tests Completed (20 pts):**
- 20: 5+ completed
- 15: 3-4 completed
- 10: 1-2 completed
- 0: 0 completed

**Support Satisfaction (15 pts):**
- 15: NPS 9-10
- 10: NPS 7-8
- 5: NPS 0-6
- 0: No interaction

**Payment Status (10 pts):**
- 10: Paid, current
- 5: Trial, engaged
- 0: Payment failed / Past due
